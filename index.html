Videostore

An android video downloader application that lets you download Shared videos from internet.

Download this project as a .zip file Download this project as a tar.gz file

Features

  • Download videos in 720p 360p 240p 144p resolutions .
  • Preview any video before downloading it. You can watch any video by streaming it directly. This prevents you from downloading wrong videos
  • Pause downloads in between and the resume them. You can even resume interrupted downloads. Saves you a lot of precious data
  • Simple User Interface that makes users to use this app extremely easy to use .
  • Can search videos with in-built voice assist search
